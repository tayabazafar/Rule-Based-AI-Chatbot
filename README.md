# 🤖 Rule-Based AI Chatbot

A beginner-friendly Rule-Based AI Chatbot built using Python as part of the DecodeLabs AI Internship.

## Features

- Greets users
- Stores the user's name
- Responds to greetings
- Provides a help menu
- Tells programming jokes
- Gives motivational quotes
- Answers basic Python and GitHub questions
- Handles unknown commands
- Exits gracefully

## Technologies Used

- Python
- Google Colab

## Concepts Used

- Variables
- Input/Output
- While Loop
- If-Else Statements
- Lists
- String Methods
- f-Strings

## How to Run

1. Open the notebook in Google Colab.
2. Run all cells.
3. Interact with the chatbot through the console.
